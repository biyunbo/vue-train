# vue-train
基于vue + vue-router + vuex + mint-ui + vue-cli 的vue版，火车票查询
> 一个火车票查询小项目，麻雀虽小，五脏俱全。

api是[途牛网](http://m.tuniu.com/train/)上找的。

## 下载
```
git clone https://github.com/biyunbo/vue-train.git
cd vue-train
# install dependencies
npm install
```

## 启动
```
npm run dev
```

## 模块
```
1.首页（日期选择）
2.城市选择
3.列表页
```
